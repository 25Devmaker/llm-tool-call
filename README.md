# LLM Tool Calling

Built in Public: Building an AI lead generation system step by step.

## Roadmap

- [x] 01 — Single tool (Groq + Google Search)
- [x] 02 — Sequential tool calling (Search → Telegram)
- [x] 03 — Cron job (runs every morning automatically)
- [x] 04 — Interactive terminal agent

## Stack
- Python
- Groq (LLM)
- SerpAPI (Google Search)
- Telegram Bot API

## Setup

1. Clone the repo
2. Create a `.env` file with your keys:
3. Install dependencies:
```bash
pip install groq google-search-results python-dotenv requests
python tool_call.py```
